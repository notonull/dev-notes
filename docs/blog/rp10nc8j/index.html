2.环境准备

2.1.离线源配置（可选）

2.1.1.挂载镜像

mount /path/to/CentOS.iso /mnt/iso

2.1.2.根据系统：配置源（dep）

## 配置deb源
deb [trusted=yes] file:/mnt/iso focal main restricted
## 重新初始化源
apt update

2.1.3.根据系统：配置源（rpm）

## 配置yum源
vim /etc/yum.repos.d/CentOS-Media.repo：
## 输入保存
[c7-media]
name=CentOS-Media
baseurl=file:///mnt/iso
enabled=1
gpgcheck=0
## 重新初始化源
yum clean all && yum makecache
